## Use
Before use it, install python library

random, shutil, numpy, matplotlib, Pillow

```zsh
pip install random

pip install pytest-shutil

pip install numpy

pip install matplotlib

pip install Pillow
```

Please run init.py

After you can enjoy Generate Image with GA!
